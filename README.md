# dotnet-microservices-sample

Enterprise-grade .NET 8 Microservices sample project demonstrating scalable backend architecture using ASP.NET Core Web API, Entity Framework Core, SQL Server, Docker, and Clean Architecture principles.

---

## Architecture Overview

This project demonstrates:

* Microservices Architecture
* RESTful APIs
* Clean Architecture
* Entity Framework Core
* SQL Server Integration
* Swagger/OpenAPI Documentation
* Docker-ready Setup
* Scalable Backend Design
* Cloud-ready Development Approach

---

## Technology Stack

| Technology            | Usage             |
| --------------------- | ----------------- |
| .NET 8                | Backend Framework |
| ASP.NET Core Web API  | REST APIs         |
| Entity Framework Core | ORM               |
| SQL Server            | Database          |
| Swagger               | API Documentation |

## Getting Started

### Clone Repository

```bash
git clone https://github.com/kirandv/dotnet-microservices-sample.git
```

### Navigate to Project

```bash
cd dotnet-microservices-sample
```

### Run Project

```bash
dotnet run
```
